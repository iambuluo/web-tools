工具介绍

Base64 是一种用 64 个字符来表示任意二进制数据的方法。常用于在 HTTP 环境下传递较长的标识信息或编码图片、文件等。

主要功能

  • 实时编码和解码转换
  • 支持 URL 安全模式（替换 +/ 为 -_）
  • 支持拖拽文件进行编码
  • 一键复制结果到剪贴板
  • 完全离线工作，保护隐私

使用场景

  • 编码图片用于 Data URL
  • 在 URL 中传递二进制数据
  • HTTP Basic 认证
  • 邮件附件编码
  • 配置文件中的二进制数据
