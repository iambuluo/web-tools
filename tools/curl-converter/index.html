CURL 转代码工具

把 curl 命令转换成 JavaScript / Python / Java / Go 请求代码

关于这个工具

在浏览器中直接解析 curl 命令，自动生成可运行的 JavaScript 与 Python 请求代码，帮助你快速复用抓包请求。

功能特点

  • 解析 Copy as cURL 输出，识别 method、headers、body 等关键字段
  • 生成 JavaScript fetch、Python requests、Java HttpClient、Go net/http 代码
  • 自动检测 JSON、表单、multipart 请求
  • 内置示例与一键复制
  • 全程本地运行，保护请求隐私

使用场景

  • 快速把抓包请求还原成脚本代码
  • 复制浏览器请求到爬虫/接口调试脚本
  • 排查 header、鉴权与 body 格式
