哈希生成器

计算文本的 MD5、SHA-1、SHA-256 等哈希值

关于这个工具

哈希生成器可以计算任意文本的多种哈希值，包括 MD5、SHA-1、SHA-256、SHA-384、SHA-512 以及 Base64 编码。使用浏览器原生 Web Crypto API，完全在本地运行。

功能特点

  • 支持 MD5 哈希计算
  • 支持 SHA-1、SHA-256、SHA-384、SHA-512 哈希计算
  • 支持 Base64 编码
  • 实时计算，即时显示结果
  • 一键复制任意哈希值
  • 完全在浏览器本地运行

使用场景

  • 生成文件校验和
  • 密码哈希存储
  • 数据完整性验证
  • API 签名生成
  • 安全令牌生成
